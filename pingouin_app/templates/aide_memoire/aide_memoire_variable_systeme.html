Secret Key et autres variable à cacher

Il faut cacher des clé des passwords des users. Mais je veux mettre mon code sur Git Alors comment faire?
crée un fichier .env et y mettre en clair vos clé. exemple : SECRET_KEY = 1234
Conseil: mettre une meilleure clé que 1234
Ensuite avec la biliothéque OS vous pouvez demandé une clé dans la variable d'environnement du server.

from dotenv import load_dotenv
import os

load_dotenv()

#recup clé secrete dans un fichier .env
SECRET_KEY = os.getenv("SECRET_KEY")
Il ne vous reste plus qu'à ajouter une variable d'environnemt dans votre serveur.
sur Heroku vous le trouverer dans setting reveal config vars
