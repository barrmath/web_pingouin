Aide Mémoire AWS_CDK

Aws cdk permets de faire de Infracstrure As A Code.
Grâce à du code python, vous pouvez créer des infracstrure dans aws.

Installation et configuration aws_cdk
prérequie: Il faut crée un compte et un role IAM qui permette de créer les instances sur aws.
Il faut aussi installer aws CLI (aws en ligne de commande).
Première étape se crée un dossier pour stocker IAAC

cdk init app -- language python
Permet d'initialiser un repertoire. (effectue aussi un depot git local)
crée un environnement virtuel
python -m venv venv
source /venv/bin/activate
pip install -r requirements.txt
si ce n'est pas déja fait : configurer son compte aws pour CLI :
aws configure ()
Pour le reste de ce tuto on va directement sur le git de AWS
